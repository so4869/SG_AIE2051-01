# Counting characters

For a given string str, count the number of each character then print every character with the number of occurrences.

For conversion between characters and integers, refer to the following code.
```c++
cout << (char) 68 << endl;
cout << (int) 'A' << endl;
```

**Input**
```
Acbc 123
```

**Output**
```
' ':1
'1':1
'2':1
'3':1
'A':1
'b':1
'c':2
```
