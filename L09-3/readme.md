# Class Template

In this task, please make a class template for class Node.  
Like function template, we will declare a template right before the class declaration:
```c++
template <typename T>
class Node{
...
};
```

Then, type parameter T can be used as a datatype in the class body.  
Please substitute datatype related to variable data with type parameter T.  
Note that if “Node” is automatically replaced with “Node”.
