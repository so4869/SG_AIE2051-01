# Ice Cream Flavors

Sarah asks her classmates for their favorite ice cream flavors to create a list. The completed list naturally contains duplicates.  
Let's create a list without duplicates using **the 'set' container.**

**Input**  
```
5
Mint
Choco
Mint
Vanilla
Peach
```

**Output**
```
4
Choco
Mint
Peach
Vanilla
```
