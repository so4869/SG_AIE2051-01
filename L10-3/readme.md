# Word Replacement

Please substitute all source words `word1` in text `str` to the target word `word2`.

Use methods of string such as `find()`, `append()`, `erase()`, `replace()` to implement word replacement. Please refer to https://cplusplus.com/reference/string/string/ for the usage.


**Input example**
```
SOGANG,ILOVESOGANG GANG NHEUNGMIN
```

**Output example**
```
SONHEUNGMIN,ILOVESONHEUNGMIN
```
